A Small World

Colonize an asteroid - create a sustainable ecosystem

      1. Build some buildings
      2. Press Colonize

If any building in the base meet its requirement for production it will supply all of the base - including all buildings - with that resource.

Experiment with different building placements to increase production and gain powerful bonuses.

The more resources the base is producing, the longer sustainability of the colony.
